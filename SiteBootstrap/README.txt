Veuillez, s'il vous plaît, ouvrir ce site via un serveur Apache. Le résultat du traitement via le php est affiché
via une autre page.

J'ai rajouté du JavaScript dans "scroll-animate.js" pour avoir l'effet de glissement avec mon bouton qui se trouve 
dans mon carousel-item.
J'ai implémenté une balise me permettant de faire  des "animation on scroll", cela permet de détecter quand
l'utilisateur arrive au niveau de l'élément sur la page. le style de l'animation est ensuite géré par moi-même
dans mon fichier CSS.

Le site est entièrement responsive. Le Carousel l'est aussi cependant la taille est géré en fonction des images
pour ne pas avoir un rendu "étiré" à cause des dimensions des images.

Tout les icones de ce site ne proviennent pas de "Glyphicons". J'ai choisi de prendre des images sur internet,
de les modifier via "Gimp" pour ensuite les redimensionner et les implémenter dans mon site. Tout les icones,
sauf l'icone principal du site, sont cliquables redirigeant vers un autre site.

Youtube et twitter offre la possibilité d'implémenter une vidéo ainsi qu'un tweet provenant de leur site.

le footer reste en bas de page peut importe le contenu du site (du main).
